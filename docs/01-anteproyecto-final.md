# Anteproyecto 1 - DAW

## Titulo

Sistema de Gestion de Inventario para el Laboratorio de Quimica del Instituto Leonardo Da Vinci.

## Participantes y roles

- Said Oudrhich: GitHub, base de datos e integracion inicial con Insforge.
- Mario Gonzalez: desarrollo frontend.
- Cesar Sanchez: desarrollo backend.

## Definicion del proyecto

El proyecto consiste en desarrollar una aplicacion web para digitalizar y centralizar el inventario del laboratorio de quimica. La plataforma permitira registrar material, equipos y reactivos, junto con sus movimientos de entrada, salida y traslado.

El objetivo principal es lograr un control real del stock, mejorar la trazabilidad y evitar perdidas o desabastecimiento.

## Justificacion

1. Necesidad detectada: se requiere sustituir el control manual por una gestion digital unificada y auditable.
2. Viabilidad tecnica y formativa: el proyecto aplica competencias de DAW en frontend, backend, base de datos y despliegue.
3. Impacto esperado: mejora operativa del laboratorio y demostracion de capacidades profesionales del equipo.

## Tecnologias definitivas

- Frontend: React + TypeScript
- Backend: Laravel (version actual)
- Base de datos: PostgreSQL en Insforge
- Despliegue: pendiente de validacion final (incluyendo opcion previa awardspace)

## Alcance inicial (MVP)

- Catalogo de inventario por categoria y ubicacion.
- Registro de movimientos y logs historicos.
- Usuarios con roles basicos.
- Importacion inicial desde dos hojas Excel ya disponibles.
